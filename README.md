### What I learned

I used Chris Burnell's [clamp tool](https://chrisburnell.com/clamp-calculator/) to calculate viewport-based clamped values.


```scss
 &__content {
    padding: clamp(2.375rem, 2.155rem + 0.939vw, 3rem) clamp(1.5rem, 0.972rem + 2.254vw, 3rem);
  }
```
